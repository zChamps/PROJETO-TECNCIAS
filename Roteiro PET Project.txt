3) Criar uma outra classe concreta que herde de ImagemAnimada e represente algo que será movimentada automaticamente. Baixe uma nova imagem na Internet para esta classe.
3.1) Você precisará implementar o método run() nesta classe. Esse método vai definir o comportamento.

private Thread t = new Thread(this);
	
public MinhaImagemAnimada() {
	t.start();
}

public void run() {
  while(true) {
    this.setX(this.getX() + 10);
    try {
	t.sleep(500);
    } catch (InterruptedException e) {
	e.printStackTrace();
    }
  }
}
